# Política de privacidade

Última atualização: 16 de julho de 2026.

## Resumo

O VisuAlg Web executa o pseudocódigo localmente. O código-fonte digitado, as
entradas do console, as abas e as variáveis não são enviados pelo aplicativo a
um servidor do projeto. Esta distribuição não inclui conta, publicidade,
telemetria nem ferramenta de analytics habilitada.

## Dados armazenados no dispositivo

A versão Web usa o armazenamento local do navegador para preferências,
autosave, recuperação de abas e dados do comando `arquivo`. Esses dados ficam
associados à origem do site e podem ser apagados pelas configurações do
navegador.

A versão Electron guarda preferências no perfil do aplicativo. O comando
`arquivo` grava arquivos de texto somente dentro da subpasta `visualg-data` da
pasta de dados do VisuAlg Web. Desinstalar o aplicativo pode não remover esses
dados automaticamente.

## Serviços externos

Ao acessar uma implantação Web, o provedor de hospedagem e a infraestrutura de
rede podem registrar dados técnicos usuais, como endereço IP, horário,
user-agent e URL solicitada, conforme as políticas deles. Ao abrir uma issue ou
relato de segurança no GitHub, o GitHub processa os dados enviados segundo sua
própria política. Não inclua dados pessoais em exemplos públicos.

## Rede e permissões

O aplicativo Electron empacotado bloqueia novas janelas, navegação externa e
solicitações de permissões por padrão. A interface não precisa enviar o conteúdo
dos algoritmos à rede para interpretar ou executar programas.
